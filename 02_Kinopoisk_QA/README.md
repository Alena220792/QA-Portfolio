# Проект: Комплексное тестирование главной страницы и личного профиля сервиса «Кинопоиск»

Проект включает в себя функциональное тестирование интерфейса, интеграционное тестирование API (серверная валидация), а также аудит производительности (Performance QA).

## 📌 Краткое описание продукта
*   **Продукт:** Сервис «Кинопоиск» (Главная страница и личный профиль пользователя).
*   **User Story:** Предоставить пользователям возможность искать и оценивать фильмы, управлять профилем (просматривать историю, добавлять фильмы в список «Буду смотреть», читать рецензии).
*   **Заказчик:** ОАНО “ДПО “СКАЕНГ”.

---

## 🛠 Выполненные виды тестирования
*   Smoke-тестирование (критический путь поиска и кастомизации профиля)
*   Функциональное тестирование UI/UX
*   Тестирование производительности (Performance Testing)
*   Тестирование API (валидация входящих параметров методом GET)

## 📊 Заключение и аналитический отчет (QA Sign-off)

**Решение о выпуске (Release Decision):** 
Выпуск продукта в продакшн **возможен**, но с условием обязательного исправления дефектов API в следующем патче. Обнаруженные проблемы серверной валидации и обработки ошибок не блокируют действия пользователей на UI, но делают бэкенд уязвимым для невалидных данных.

**Анализ влияния дефектов на пользователей:**
1.  *Дефекты API (валидация):* Не были выявлены на этапе функционального UI-тестирования, так как интерфейс ограничивает ввод. Напрямую на пользовательский опыт (UX) не влияют, но требуют исправления со стороны бэкенда.
2.  *Задержка плеера (более 3 секунд):* Замедляет запуск видео, что негативно сказывается на лояльности пользователей. Требуется оптимизация кэширования или сжатия потока.

**Рекомендации:**
1.  Разработчикам Бэкенда: Настроить корректные HTTP-ответы (например, `404 Not Found` или `400 Bad Request`) при отправке некорректных поисковых запросов через API.
2.  Разработчикам Фронтенда: Оптимизировать скрипты инициализации плеера для сокращения времени загрузки до лимита в 1.5–2 секунды.
3.  Разработать автоматизированные скрипты для регулярной проверки валидации API эндпоинтов.
